[agent-browser](https://github.com/vercel-labs/agent-browser). A browser automation CLI built for AI agents. I started looking for browser automation tools and skills that generally enhance coding agents' capabilities. First I recalled [Playwright MCP](https://github.com/microsoft/playwright-mcp), a tool I already knew about but hadn't tried. Its README recommended the CLI + skills pair for this generation of coding agents, [Playwright CLI](https://github.com/microsoft/playwright-cli). Makes sense, I thought.

Then I remembered a similar tool by the Vercel team called agent-browser, and I found it more popular (by stars). There are several agent skills written for it, one of which even points out that the tool can control any Electron-based apps, like VS Code and Slack.

I installed it for all my agents (Amp, Codex, and Claude Code for now). Here's an example on Amp: [Agent-browser capabilities and example](https://ampcode.com/threads/T-019da3cc-8b6d-75bf-a8b6-1fdb8880936f).
