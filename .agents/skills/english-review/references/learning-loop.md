# Learning Loop

Use this file when English review is part of practice, not just proofreading.

## Modes

Choose the lightest mode that matches the request.

### Direct review

- Fix grammar, wording, and awkward phrasing
- Keep comments brief
- Do not turn a quick pass into a lesson unless the user asks for it

### Comparison review

- Compare the author's draft with a reference version
- Focus on reusable differences in expression, sentence pattern, paragraph flow, or voice
- Explain why a rewrite is more natural, more precise, or simply better suited to the piece

### Synthesis review

- Help the author produce a final version after comparison
- Preserve the author's meaning and preferred voice
- Borrow stronger phrases or structures selectively instead of replacing everything

## Preferred output

1. `What already works`
2. `What to improve`
3. `Better patterns to reuse`
4. `Possible final version` if the user asks for synthesis

## Guardrails

- Do not assume the AI reference is always better
- Do not erase the author's voice
- Do not over-explain small edits in a quick review
